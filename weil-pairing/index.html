Elliptic curves demo Weil pairing

Definition

Let E(K) be some elliptic curve over a field K, Q, R \in E(K)[n], where E(K)[n] = \{T \in E(K) | n \cdotp T = O \}. Then weil-pairing is a function that produces the n-th root of unity:

w(Q, P) \in \mu_n,

where \mu_n = \{x \in K | x^n = 1\}

Properties

  • bilinearity:
    w(Q_1 + Q_2, P) = w(Q_1, P)w(Q_2, P)
    w(Q, P_1 + P_2) = w(Q, P_1)w(Q, P_2)
  • alternating property:
    w(Q, P) = w(P, Q)^{-1}
  • nondegenerate property:
    \forall P\ w(Q, P) = 1 \Leftrightarrow Q = O
    \forall Q\ w(Q, P) = 1 \Leftrightarrow P = O
  • w(Q, Q) = 1

Building using divisors

Consider the function f(x, y): E(K) \rightarrow \mathbb{R}. We define it's divider as following:

div(f) = \sum_i m_i(Z_i) - \sum_j m_j(P_j),

where P are poles of f on E(K), Z are zeros and m_i, m_j are corresponding multiplicities. Note, that (P) and (Q) formally are elements of new group and cannot be added like points on curve. Function f is called rational if it can be represented as ratio of two polynomials. For rational functions \sum_i m_i - \sum_j m_j = 0 (simple heuristic approach is just to pad missing poles with O, thus for line intersecting E(K) in points A, B, C (obviously rational function), divider will be (A) + (B) + (C) - 3(O)). Let's take S, T \in E(K)[m] for some m coprime to p and construct two divisors A = (S) - (O), B = (T) - (O). Now we should find two functions f_A, f_b:\ div(f_A) = mA,\ div(f_B) = mB, and the weil-pairing will be defined as:

w(S, T) = \frac{f_A(B)}{f_B(A)}.

It can be shown that weil-pairing doesn't depend on the choice of A, B and the corresponding functions.

Miller's algorithm to compute weil-pairing

Schema

Suppose we want to calculate weil-pairing for S, T \in E(K). Choose S', T' from E(K) such that all T, S, T+T', S+S' are different. Set A = (S + S') - (S'), B = (T + T') - (T') (trick to avoid infinity).
